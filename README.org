#+TITLE: yasnippets

Snippets for org-mode. See [[https://github.com/capitaomorte/yasnippet][Yasnippet in Github]].

They have bindings ~C-c y~ plus an extra letter.

| name        | key | letter |
|-------------+-----+--------|
| corollary   | cor | c      |
| definition  | def | d      |
| equation    | eqn | e      |
| lemma       | lem | l      |
| proof       | prf | f      |
| proposition | pro | p      |
| theorem     | thm | t      |
